# University Ranking Data Analysis

Python project for analyzing university rankings and capitals datasets using CSV handling and reusable functions

## Features 
- Python script (`ranks.py`) for ranking and calculations 
- Jupyter notebook (`minor_project_1.ipynb`) with step-by-step analysis 
- Organized outputs aligned with sample test cases


## project structure
```
mini_project-1/
├ranks.py
├ranking_demo.ipynb
├Readme.md
├.gitignore
└data/
├capitals.csv
├universities.csv
```

## requirements
- python 3.13+

## usage
run the main script:

```bash 
python ranks.py
```
or explore jupyter notebook

```bash
jupyter notebook ranking_demo.ipynb
```
